Política de Privacidade

1. Informações Coletadas

A Naruto BR API não coleta dados pessoais de usuários. A API fornece apenas informações públicas sobre o universo de Naruto, acessíveis através de endpoints de consulta.

2. Uso de Dados

Todos os dados disponibilizados são de natureza pública e relacionados ao universo do anime e mangá Naruto. Não há coleta, armazenamento ou processamento de informações pessoais de usuários.

3. Cookies e Rastreamento

Não utilizamos cookies ou qualquer tecnologia de rastreamento. A API é projetada para ser simples, direta e focada em fornecer informações sobre Naruto.

4. Segurança

Implementamos medidas de segurança padrão para proteger nossa infraestrutura e garantir a disponibilidade e integridade dos dados da API.

5. Alterações na Política

Reservamo-nos o direito de modificar esta política de privacidade a qualquer momento. Recomendamos a revisão periódica desta página.
